# React Portfolio Website

This is a personal portfolio website built using React to showcase my projects, skills, and experience.

## Features
- About Me section
- Projects section
- Skills section
- Resume section
- Contact section
- Theme toggle (Light/Dark)
- Crypto Station project showing live cryptocurrency prices using Fetch API and Promises

## Technologies Used
- React
- Context API
- JavaScript (ES6)
- HTML5
- CSS3
- Fetch API
- Git & GitHub
- Vercel

## Development Process
- Created reusable React components
- Used Context API for theme management
- Implemented API calls using fetch and promises
- Styled the application using CSS
- Ensured responsive design for mobile and desktop

## Production Optimization
- Built production-ready app using `npm run build`
- Used lazy loading for better performance
- Optimized images and assets

## Deployment
- Pushed project to GitHub
- Connected GitHub repository to Vercel
- Deployed using Create React App preset
- Enabled SSL automatically via Vercel

## Challenges Faced
- Understanding Context API structure  
  **Solution:** Practiced with small examples and applied it to theme toggling

- Deployment configuration issues  
  **Solution:** Corrected framework preset and build folder in Vercel
